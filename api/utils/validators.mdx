## Overview

Django provides a comprehensive set of validators to validate data in forms and model fields. All validators raise `ValidationError` when validation fails.

## Email Validators

### EmailValidator

Validates that a value is a valid email address.

```python
from django.core.validators import EmailValidator

validator = EmailValidator(
    message=None,
    code=None,
    allowlist=None
)
```

<ParamField path="message" type="str" default="'Enter a valid email address.'">
  Custom error message for validation failures
</ParamField>

<ParamField path="code" type="str" default="'invalid'">
  Error code for ValidationError
</ParamField>

<ParamField path="allowlist" type="list" default="['localhost']">
  List of allowed domain names that bypass domain validation
</ParamField>

**Usage:**
```python
from django.core.validators import EmailValidator, validate_email

validator = EmailValidator()
validator('user@example.com')  # Passes
validator('invalid-email')     # Raises ValidationError

# Pre-configured instance
validate_email('user@example.com')
```

## URL Validators

### URLValidator

Validates that a value is a valid URL.

```python
from django.core.validators import URLValidator

validator = URLValidator(
    schemes=None,
    regex=None,
    message=None,
    code=None,
    inverse_match=None,
    flags=None
)
```

<ParamField path="schemes" type="list" default="['http', 'https', 'ftp', 'ftps']">
  List of allowed URL schemes
</ParamField>

<ParamField path="message" type="str" default="'Enter a valid URL.'">
  Custom error message
</ParamField>

<ParamField path="code" type="str" default="'invalid'">
  Error code for ValidationError
</ParamField>

**Usage:**
```python
from django.core.validators import URLValidator

validator = URLValidator()
validator('https://example.com')  # Passes
validator('not a url')            # Raises ValidationError

# Custom schemes
validator = URLValidator(schemes=['http', 'https', 'git'])
validator('git://github.com/repo')
```

## Value Validators

### MinValueValidator

Validates that a value is greater than or equal to a limit.

```python
from django.core.validators import MinValueValidator

validator = MinValueValidator(
    limit_value,
    message=None
)
```

<ParamField path="limit_value" type="int | float | callable" required>
  Minimum value allowed (can be a callable that returns the limit)
</ParamField>

<ParamField path="message" type="str" default="'Ensure this value is greater than or equal to %(limit_value)s.'">
  Custom error message
</ParamField>

**Usage:**
```python
from django.core.validators import MinValueValidator

validator = MinValueValidator(0)
validator(5)   # Passes
validator(-1)  # Raises ValidationError
```

## Regex Validators

### RegexValidator

Validates that a value matches a regular expression.

```python
from django.core.validators import RegexValidator

validator = RegexValidator(
    regex=None,
    message=None,
    code=None,
    inverse_match=None,
    flags=None
)
```

<ParamField path="regex" type="str | Pattern" default="''">
  Regular expression pattern to match
</ParamField>

<ParamField path="message" type="str" default="'Enter a valid value.'">
  Custom error message
</ParamField>

<ParamField path="code" type="str" default="'invalid'">
  Error code for ValidationError
</ParamField>

<ParamField path="inverse_match" type="bool" default="False">
  If True, validation fails if the regex matches
</ParamField>

<ParamField path="flags" type="int" default="0">
  Regular expression flags (e.g., re.IGNORECASE)
</ParamField>

**Usage:**
```python
from django.core.validators import RegexValidator
import re

validator = RegexValidator(
    regex=r'^[0-9]+$',
    message='Only digits are allowed',
    code='invalid_digits'
)
validator('12345')  # Passes
validator('abc')    # Raises ValidationError
```

## Decimal Validators

### DecimalValidator

Validates that a decimal value doesn't exceed maximum digits and decimal places.

```python
from django.core.validators import DecimalValidator

validator = DecimalValidator(
    max_digits,
    decimal_places
)
```

<ParamField path="max_digits" type="int" required>
  Maximum number of total digits allowed
</ParamField>

<ParamField path="decimal_places" type="int" required>
  Maximum number of decimal places allowed
</ParamField>

**Usage:**
```python
from decimal import Decimal
from django.core.validators import DecimalValidator

validator = DecimalValidator(max_digits=5, decimal_places=2)
validator(Decimal('123.45'))  # Passes
validator(Decimal('1234.567'))  # Raises ValidationError
```

## File Validators

### FileExtensionValidator

Validates that a file extension is in an allowed list.

```python
from django.core.validators import FileExtensionValidator

validator = FileExtensionValidator(
    allowed_extensions=None,
    message=None,
    code=None
)
```

<ParamField path="allowed_extensions" type="list" default="None">
  List of allowed file extensions (without dots)
</ParamField>

<ParamField path="message" type="str" default="'File extension is not allowed.'">
  Custom error message with `%(extension)s` placeholder
</ParamField>

<ParamField path="code" type="str" default="'invalid_extension'">
  Error code for ValidationError
</ParamField>

**Usage:**
```python
from django.core.validators import FileExtensionValidator, validate_image_file_extension

validator = FileExtensionValidator(allowed_extensions=['pdf', 'doc', 'docx'])

# Pre-configured for images
validate_image_file_extension(uploaded_file)
```

## IP Address Validators

### validate_ipv4_address

```python

validate_ipv4_address('256.0.0.1')    # Raises ValidationError
```
