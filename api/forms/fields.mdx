## Base Field Class

All form fields inherit from the base `Field` class.

```python
Field(
    required=True,
    widget=None,
    label=None,
    initial=None,
    help_text="",
    error_messages=None,
    show_hidden_initial=False,
    validators=(),
    localize=False,
    disabled=False,
    label_suffix=None,
    template_name=None,
    bound_field_class=None
)
```

<ParamField path="required" type="bool" default="True">
  Whether the field is required. If `True`, validation fails if field is empty.
</ParamField>

<ParamField path="widget" type="Widget">
  Widget instance to use for rendering. Each field has a default widget.
</ParamField>

<ParamField path="label" type="str">
  Human-readable label for the field. Auto-generated from field name if not provided.
</ParamField>

<ParamField path="initial" type="any">
  Initial value for the field in unbound forms.
</ParamField>

<ParamField path="help_text" type="str" default="''">
  Help text to display with the field.
</ParamField>

<ParamField path="error_messages" type="dict">
  Dictionary of error codes to custom error messages.
</ParamField>

<ParamField path="show_hidden_initial" type="bool" default="False">
  Whether to render a hidden widget with the initial value.
</ParamField>

<ParamField path="validators" type="list">
  List of additional validator functions to apply.
</ParamField>

<ParamField path="localize" type="bool" default="False">
  Whether to localize field input and output.
</ParamField>

<ParamField path="disabled" type="bool" default="False">
  Whether the field is disabled (shown but not editable).
</ParamField>

<ParamField path="label_suffix" type="str">
  Suffix to append to the label. Overrides form-level setting.
</ParamField>

## Text Fields

### CharField

Accepts text input.

```python
CharField(
    max_length=None,
    min_length=None,
    strip=True,
    empty_value="",
    **kwargs
)
```

<ParamField path="max_length" type="int">
  Maximum number of characters allowed.
</ParamField>

<ParamField path="min_length" type="int">
  Minimum number of characters required.
</ParamField>

<ParamField path="strip" type="bool" default="True">
  Whether to strip leading and trailing whitespace.
</ParamField>

<ParamField path="empty_value" type="str" default="''">
  Value to return for empty input.
</ParamField>

```python
name = forms.CharField(
    max_length=100,
    min_length=2,
    label="Your Name",
    help_text="Enter your full name"
)
```

### EmailField

Validates that input is a valid email address.

```python
EmailField(**kwargs)
```

Default widget: `EmailInput`  
Default max_length: `320` (per RFC 3696)

```python
email = forms.EmailField(
    required=True,
    error_messages={'invalid': 'Please enter a valid email address'}
)
```

### URLField

Validates that input is a valid URL.

```python
URLField(
    assume_scheme="https",
    **kwargs
)
```

<ParamField path="assume_scheme" type="str" default="https">
  Scheme to prepend if none provided (e.g., "https", "http").
</ParamField>

Default widget: `URLInput`

```python
website = forms.URLField(
    required=False,
    assume_scheme="https"
)
```

### SlugField

Validates input contains only letters, numbers, underscores, or hyphens.

```python
SlugField(
    allow_unicode=False,
    **kwargs
)
```

<ParamField path="allow_unicode" type="bool" default="False">
  Whether to allow Unicode letters in addition to ASCII.
</ParamField>

```python
slug = forms.SlugField(max_length=50)
```

### RegexField

Validates input against a regular expression.

```python
RegexField(
    regex,
    **kwargs
)
```

<ParamField path="regex" type="str | Pattern" required>
  Regular expression string or compiled pattern object.
</ParamField>

```python
phone = forms.RegexField(
    regex=r'^\d{3}-\d{3}-\d{4}$',
    error_messages={'invalid': 'Enter format: XXX-XXX-XXXX'}
)
```

IntegerField(

DateField(

birthdate = forms.DateField(

appointment_time = forms.TimeField(

DateTimeField(

## Choice Fields

### ChoiceField

Validates that input is one of a set of valid choices.

```python
ChoiceField(
    choices=(),
    **kwargs
)
```

<ParamField path="choices" type="list" required>
  List of 2-tuples `(value, label)` or nested groups.
</ParamField>

Default widget: `Select`

```python
STATUS_CHOICES = [
    ('draft', 'Draft'),
    ('published', 'Published'),
    ('archived', 'Archived'),
]

status = forms.ChoiceField(
    choices=STATUS_CHOICES,
    initial='draft'
)
```

### TypedChoiceField

Like ChoiceField but coerces the value to a specific type.

```python
TypedChoiceField(
    choices=(),
    coerce=lambda val: val,
    empty_value="",
    **kwargs
)
```

<ParamField path="coerce" type="callable" default="lambda val: val">
  Function to coerce the value to the desired type.
</ParamField>

<ParamField path="empty_value" type="any" default="''">
  Value to return for empty input.
</ParamField>

```python
rating = forms.TypedChoiceField(
    choices=[(1, 'One'), (2, 'Two'), (3, 'Three')],
    coerce=int
)
```

### MultipleChoiceField

Validates that input is one or more of valid choices.

```python
MultipleChoiceField(
    choices=(),
    **kwargs
)
```

Default widget: `SelectMultiple`  
Returns: List of strings

```python
TAG_CHOICES = [
    ('python', 'Python'),
    ('django', 'Django'),
    ('javascript', 'JavaScript'),
]

tags = forms.MultipleChoiceField(
    choices=TAG_CHOICES,
    required=False
)
```

### TypedMultipleChoiceField

Like MultipleChoiceField but coerces values to a specific type.

```python
TypedMultipleChoiceField(
    choices=(),
    coerce=lambda val: val,
    empty_value=[],
    **kwargs
)
```

```python
years = forms.TypedMultipleChoiceField(
    choices=[(2021, '2021'), (2022, '2022'), (2023, '2023')],
    coerce=int
)
```

## File Fields

### FileField

Validates file upload.

```python
FileField(
    max_length=None,
    allow_empty_file=False,
    **kwargs
)
```

<ParamField path="max_length" type="int">
  Maximum length of the filename.
</ParamField>

<ParamField path="allow_empty_file" type="bool" default="False">
  Whether to allow empty files (0 bytes).
</ParamField>

Default widget: `ClearableFileInput`  
Returns: `UploadedFile` object

```python
attachment = forms.FileField(
    required=False,
    help_text="Max file size: 5MB"
)
```

### ImageField

Validates that uploaded file is a valid image.

```python
ImageField(**kwargs)
```

Requires: Pillow library  
Default widget: `ClearableFileInput`  
Returns: `UploadedFile` object with image validation

```python
avatar = forms.ImageField(
    required=False,
    help_text="Upload a profile picture"
)
```

### FilePathField

Allows selection of files from server filesystem.

```python
FilePathField(
    path,
    match=None,
    recursive=False,
    allow_files=True,
    allow_folders=False,
    **kwargs
)
```

<ParamField path="path" type="str" required>
  Absolute filesystem path to directory.
</ParamField>

<ParamField path="match" type="str">
  Regular expression pattern to filter filenames.
</ParamField>

<ParamField path="recursive" type="bool" default="False">
  Whether to include subdirectories.
</ParamField>

<ParamField path="allow_files" type="bool" default="True">
  Whether to include files.
</ParamField>

<ParamField path="allow_folders" type="bool" default="False">
  Whether to include folders.
</ParamField>

```python
template = forms.FilePathField(
    path="/path/to/templates/",
    match=r'\.html$',
    recursive=True
)
```

## Composite Fields

### ComboField

Validates input against multiple fields sequentially.

```python
ComboField(
    fields,
    **kwargs
)
```

<ParamField path="fields" type="list" required>
  List of Field instances to validate against.
</ParamField>

```python
secure_password = forms.ComboField(
    fields=[
        forms.CharField(min_length=8),
        forms.RegexField(regex=r'[A-Z]', error_messages={'invalid': 'Need uppercase'}),
        forms.RegexField(regex=r'[0-9]', error_messages={'invalid': 'Need number'}),
    ]
)
```

### MultiValueField

Aggregates multiple fields into a single field.

```python
MultiValueField(
    fields,
    require_all_fields=True,
    **kwargs
)
```

<ParamField path="fields" type="list" required>
  List of Field instances.
</ParamField>

<ParamField path="require_all_fields" type="bool" default="True">
  Whether all sub-fields are required.
</ParamField>

Must implement `compress()` method to combine field values.

### SplitDateTimeField

Splits datetime input into separate date and time fields.

```python
SplitDateTimeField(
    input_date_formats=None,
    input_time_formats=None,
    **kwargs
)
```

Default widget: `SplitDateTimeWidget`  
Returns: `datetime.datetime` object

```python
event_datetime = forms.SplitDateTimeField()
```

## Field Methods

All fields inherit these methods:

### clean()

```python
field.clean(value) -> any
```

Validates and returns cleaned data. Raises `ValidationError` if invalid.

### to_python()

```python
field.to_python(value) -> any
```

Converts input value to appropriate Python type.

### validate()

```python
field.validate(value) -> None
```

Performs field-specific validation.

### run_validators()

```python
field.run_validators(value) -> None
```

Runs all validators attached to the field.

### has_changed()

```python
field.has_changed(initial, data) -> bool
```

Returns `True` if data differs from initial value.
