# Manager API

A Manager is the interface through which database query operations are provided to Django models. At least one Manager exists for every model in a Django application.

## BaseManager

The base class that all managers inherit from.

### Methods

#### get_queryset()

Returns a new QuerySet object. Subclass this method to customize the behavior of the Manager.

```python
class DahlManager(models.Manager):
    def get_queryset(self):
        return super().get_queryset().filter(author='Roald Dahl')
```

### Class Methods

#### from_queryset()

Creates a new Manager class that inherits methods from a custom QuerySet class.

```python
class PersonQuerySet(models.QuerySet):
    def authors(self):
        return self.filter(role='A')
    
    def editors(self):
        return self.filter(role='E')

PersonManager = models.Manager.from_queryset(PersonQuerySet)

class Person(models.Model):
    objects = PersonManager()
```

<ParamField path="queryset_class" type="type" required>
  The QuerySet class to inherit methods from.
</ParamField>

<ParamField path="class_name" type="str" optional>
  The name for the new Manager class. Auto-generated if not provided.
</ParamField>

<ResponseField name="return" type="type">
  A new Manager class with methods from the QuerySet.
</ResponseField>

## Manager

The default Manager class that inherits from BaseManager.

```python
class Entry(models.Model):
    # Default manager
    objects = models.Manager()
    
    # Custom manager
    published = PublishedManager()
```

**Reference:** `django/db/models/manager.py:176`

### Creating Custom Managers

You can create custom managers by subclassing Manager:

```python
class PublishedManager(models.Manager):
    def get_queryset(self):
        return super().get_queryset().filter(status='published')

class Entry(models.Model):
    title = models.CharField(max_length=200)
    status = models.CharField(max_length=20)
    
    objects = models.Manager()  # Default manager
    published = PublishedManager()  # Custom manager
```

## Manager Inheritance

Managers are inherited by child models:

```python
class BaseModel(models.Model):
    objects = models.Manager()
    custom = CustomManager()
    
    class Meta:
        abstract = True

class ChildModel(BaseModel):
    # Inherits both 'objects' and 'custom' managers
    pass
```

## Default Manager

The first Manager defined on a model becomes the default manager. You can specify a different default manager using `Meta.default_manager_name`:

```python
class Entry(models.Model):
    published = PublishedManager()
    objects = models.Manager()
    
    class Meta:
        default_manager_name = 'objects'
```

## Base Manager

You can specify which manager Django should use for relationships and other internal operations using `Meta.base_manager_name`:

```python
class Entry(models.Model):
    objects = models.Manager()
    published = PublishedManager()
    
    class Meta:
        base_manager_name = 'objects'
```

## Calling Custom QuerySet Methods on Managers

To make custom QuerySet methods available on your Manager, use `from_queryset()`:

```python
class PersonQuerySet(models.QuerySet):
    def authors(self):
        return self.filter(role='A')
    
    def editors(self):
        return self.filter(role='E')

class PersonManager(models.Manager):
    def get_queryset(self):
        return PersonQuerySet(self.model, using=self._db)

PersonManager = PersonManager.from_queryset(PersonQuerySet)

class Person(models.Model):
    first_name = models.CharField(max_length=50)
    last_name = models.CharField(max_length=50)
    role = models.CharField(max_length=1)
    
    people = PersonManager()

# Now you can call:
Person.people.authors()
Person.people.editors()
```

## Manager Names

By convention, the default Manager is named `objects`, but you can use any valid Python identifier:

```python
class Entry(models.Model):
    # ...
    entries = models.Manager()  # Custom name
    published_entries = PublishedManager()
```

**Note:** Accessing managers on model instances raises an AttributeError. Managers are only accessible via model classes:

```python
# Correct
Entry.objects.all()

# Raises AttributeError
entry = Entry.objects.get(pk=1)
entry.objects.all()  # Error!
```
